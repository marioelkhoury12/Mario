🛒 Toronto Grocery Price Compare

Setup Guide — How to Download & Open the App

What Is This?

A web app that compares grocery flyer prices across 8 Toronto stores so you can find the cheapest deals. It also lets you add coupons that automatically apply to price comparisons.

🏪8 Stores: No Frills, Loblaws, Metro, Food Basics, FreshCo, Walmart, T&T Supermarket, Sobeys
📊50+ Items: Chicken, beef, milk, eggs, bread, cheese, rice, pasta sauce, and more
🎟️Coupon Manager: Add coupons and they auto-apply to price comparisons

How to Download & Open

1 Go to the GitHub repository
Open Safari and go to:
https://github.com/marioelkhoury12/Mario
2 Switch to the correct branch
Click the branch dropdown (says main) and select:
claude/grocery-price-comparison-T4ydX
3 Open the file
Click on grocery-compare.html in the file list
4 Download the raw file
Click the Raw button (top right of the file view).
Then press ⌘ Cmd + S to save the file.
5 Find the downloaded file
Open Finder and go to your Downloads folder.
Shortcut: press ⌥ Option + ⌘ Cmd + L in Finder.
Or click the ↓ download arrow in Safari's top-right toolbar.
6 Open in Safari
Double-click the file grocery-compare.html
It will open in Safari and the app is ready to use!
✅ No installation needed!
This is a single self-contained file. No server, no Node.js, no npm — just open it in your browser.

How to Use the App

🔍 Price Compare Tab
Type a product name (e.g. "Chicken Breast") and click Compare Prices. Results are ranked cheapest first with a green "Best Price" badge.
📋 Browse Flyers Tab
See all flyer items in a grid. Use the dropdowns to filter by store, category, or sale items. Use the search box to find specific products.
🎟️ My Coupons Tab
Click "+ Add Coupon" to enter a coupon. Choose the store, product, discount type ($ or %), and value. Coupons are automatically applied when you compare prices!
➕ Add Item Tab
Spot a deal in a flyer? Add it manually here. Pick the store, enter the product name, price, and category. It will appear in all searches.
💡 Your data is saved in your browser
All coupons and items you add are saved in Safari's local storage. They will persist between sessions, but clearing Safari's website data will erase them.

Troubleshooting

File opens as text instead of a web page?
Right-click the file → Open With → Safari
Can't find the downloaded file?
In Safari: click Safari menu → Settings → General → check "File download location" to see where files are saved.
